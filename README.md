# MarkAny - 文档双向转换工具

一款支持 PDF、Word、Excel、PowerPoint、HTML 等格式与 Markdown 双向转换的桌面应用。

## 功能特点

- 多格式支持：PDF、Word、Excel、PowerPoint、HTML、TXT ↔ Markdown
- 高质量转换：保留原文档结构、表格、图片、代码块
- OCR 支持：扫描件 PDF 也能识别转换
- 批量处理：一次添加多个文件批量转换
- 简洁界面：拖放操作，开箱即用

## 安装

下载 `MarkAny-Setup-1.0.0.exe` 运行安装即可，无需配置任何环境。

## 使用方法

1. 选择源格式和目标格式
2. 拖放或点击选择文件
3. 设置输出目录
4. 点击"开始转换"

## 系统要求

- Windows 10/11 (64位)
- 约 500MB 可用磁盘空间
