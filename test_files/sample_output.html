MarkAny 转换器测试文档

项目简介

MarkAny 是一个强大的文档格式转换工具，支持多种文件格式之间的互相转换。

功能特点

  • 多格式支持：PDF、Word、Excel、PPT、HTML、Markdown
  • 批量转换：支持一次性转换多个文件
  • 格式保持：尽量保持原始文档的排版格式

代码示例

import sys
sys.path.insert(0, 'src/python')
from converters.pdf_converter import PDFConverter

converter = PDFConverter()
result = converter.to_markdown('test.pdf', './output')
